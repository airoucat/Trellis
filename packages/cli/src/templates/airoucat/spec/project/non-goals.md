# Non-Goals

Airoucat Trellis mode does not:

- Replace Trellis task schema.
- Use graphify as a source of truth.
- Automatically commit, push, deploy, or publish.
- Execute destructive work from vague wording.
- Put every project rule into one giant AGENTS.md file.
- Call an LLM from hooks to classify intent.

Hooks provide facts. The AI session makes the semantic decision.
