# Review Checklist

Use this before finish:

- [ ] Scope matches the task.
- [ ] No unrelated files changed.
- [ ] Public behavior is covered by tests or equivalent verification.
- [ ] Runtime behavior has runtime evidence when relevant.
- [ ] Evidence.md records commands and results.
- [ ] Graphify was rebuilt if enabled and code changed.
- [ ] Specs were updated when behavior or conventions changed.
- [ ] Hostile review has no blocking findings.
